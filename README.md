## 🔧 설치 및 실행 방법

**(1) 레포지토리 클론** 
```
git clone https://github.com/BellyWelly/BellyWelly-FE.git
```
**(2) 필요한 패키지 설치** 
```
npm install
```
**(3) 실행** 
```
npm run start
```
**(4) 빌드** 
```
npm run build
```
